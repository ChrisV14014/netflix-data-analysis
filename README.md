# Netflix Data Analysis

This project analyzes a dataset from Netflix to explore trends in movies and TV shows.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Netflix content increased significantly after 2015
- Movies are more common than TV shows
- The United States produces the most content
- Drama and international genres are very popular
